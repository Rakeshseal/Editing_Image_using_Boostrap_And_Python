Editing Pics - Edit Your Image online

About us

Editing Pics is a image editing web project. This project will help to quickly convert images of different formats to JPG, PNG, Webp and Grayscale format. The web is made on Python, Html and Boostrap only. And the web page is designed on Figma software. And this project is still ongoing with efforts to improve it so that it is very beneficial for the users.

Help

First click the Choose File button then select your image (only one). Next Choose an Operation like Jpg, Png, Webp, Grayscale then click the Submit button. Next you receive a notification and click "here".
